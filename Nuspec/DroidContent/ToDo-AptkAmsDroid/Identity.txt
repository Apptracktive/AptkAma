//// Setup ////

1. (Optional) If you plan to handle expired authentication token automaticaly, add these lines BEFORE AptkAmsPluginLoader.Init(configuration, ApplicationContext); :

	var identityHandler = new AptkAmsIdentityHandler(configuration);
    configuration.HttpHandlers = new HttpMessageHandler[] { identityHandler };

and AFTER AptkAmsPluginLoader.Init(configuration, ApplicationContext); add this line:

	identityHandler.AptkAmsService = AptkAmsPluginLoader.Instance;

Note that you can provide a callback action if user is logged out like navigating to the login view.
